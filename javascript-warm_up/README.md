# JavaScript Warm-Up

> Short Node.js exercises to practice syntax, output, strings, and arrays.

## Description & Objectives

This project contains the first JavaScript tasks used to practice basic language features in a Node.js environment. The goal is to write small, focused scripts and learn how to structure output clearly.

## Technologies Used

| Tool | Purpose |
| --- | --- |
| Node.js | Runtime used to execute the scripts |
| JavaScript | Main language for the exercises |

## Prerequisites

- Linux, macOS, or Windows with a terminal that can run Node.js
- Node.js 16 or newer
- No third-party npm dependencies are required

## Installation

No installation step is needed. Clone the repository and move into the project folder.

```bash
cd javascript-warm_up
```

## Usage

Run each script with Node.js:

```bash
node 0-javascript_is_amazing.js
node 1-multi_languages.js
```

Example output:

```text
JavaScript is amazing
C is fun
Python is cool
JavaScript is amazing
```

## Features

- Simple string output with `console.log`
- Practice with Node.js script execution
- Clean, short scripts that are easy to test manually
