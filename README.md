# Workspace Overview

> A small collection of Holberton-style Python and JavaScript exercise projects.

## Projects

| Project | Language | Purpose |
| --- | --- | --- |
| `javascript-warm_up` | JavaScript | Node.js warm-up exercises for strings, arrays, and console output. |
| `python-object_relational_mapping` | Python | MySQL and SQLAlchemy ORM practice. |
| `python-serialization` | Python | JSON, pickle, CSV, and XML serialization exercises. |
| `restful-api` | Python | HTTP, Flask, and API security exercises. |
| `holbertonschool-core-engineering` | Python / documentation | Core engineering learning materials and environment demos. |

## Documentation Standard

- Python modules, functions, and classes use [PEP 257](https://peps.python.org/pep-0257/) docstrings.
- JavaScript scripts use concise module comments and, when functions are present, JSDoc-style documentation.

## General Requirements

- Python 3.10+ for the Python exercises.
- Node.js 16+ for the JavaScript exercises.
- MySQL Server and client libraries for ORM exercises.
- Git-aware hygiene through the repository-level `.gitignore`.

## Notes

- The repository is organized as a set of standalone training projects rather than a single application.
- Most tasks are CLI-oriented and can be executed directly from their project folders.
