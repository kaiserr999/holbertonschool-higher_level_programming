# JavaScript DOM Manipulation

> Learn how to select, update, and react to DOM elements in the browser.

## Description & Objectives

This project introduces DOM manipulation with JavaScript. It focuses on selecting elements, changing styles and content, handling events, and integrating API data into the page.

## Technologies Used

| Tool | Purpose |
| --- | --- |
| JavaScript | Main language for DOM logic |
| Browser DOM APIs | Element selection, events, and rendering |
| HTML / CSS | Structure and presentation of the page |

## Prerequisites

- A modern browser
- A text editor or VS Code
- Node.js is useful for local tooling, but the exercises run in the browser

## Installation

Clone the repository and open the project folder in your editor or browser-based environment.

## Usage

Open the HTML page in a browser and run the scripts from the exercise instructions.

Example interaction:

```text
Click a button to change the page title, text, or styles.
```

## Features

- DOM selection with `querySelector` and related APIs
- Style and content updates
- Event handling for user interaction
- API-driven content updates
