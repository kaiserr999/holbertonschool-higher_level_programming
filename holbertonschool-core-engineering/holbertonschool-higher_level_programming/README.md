# Holbertonschool Higher Level Programming

> Higher-level programming exercises and supporting examples for the Holberton curriculum.

## Description & Objectives

This folder groups higher-level programming training materials. It currently contains a JavaScript warm-up subset and acts as a parent workspace for related exercises.

## Technologies Used

| Tool | Purpose |
| --- | --- |
| JavaScript | Exercise language for the warm-up set |
| Node.js | Runtime used to execute the scripts |

## Prerequisites

- Node.js 16 or newer
- A terminal for running JavaScript scripts

## Installation

No installation is required. Open the folder and run the scripts directly.

## Usage

```bash
node javascript-warm_up/0-javascript_is_amazing.js
node javascript-warm_up/1-multi_languages.js
```

## Features

- Small JavaScript exercises
- Console output practice
- Simple folder-based organization for curriculum tasks

## Documentation Standard

JavaScript files should keep concise module comments and readable function names when helpers are introduced.

## Contributions & Thanks

Part of the Holberton School higher-level programming path. Keep changes aligned with the exercise style and naming used by the curriculum.
