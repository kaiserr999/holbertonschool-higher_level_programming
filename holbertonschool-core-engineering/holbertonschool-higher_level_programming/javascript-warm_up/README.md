# JavaScript Warm-Up

> JavaScript starter exercises for output, strings, and basic syntax.

## Description & Objectives

This subset mirrors the JavaScript warm-up exercises used to practice Node.js script execution and simple console output.

## Technologies Used

| Tool | Purpose |
| --- | --- |
| JavaScript | Main language for the scripts |
| Node.js | Runtime used to run the exercises |

## Prerequisites

- Node.js 16 or newer
- A terminal to execute the scripts

## Installation

No installation step is needed. Open the folder and run the scripts directly.

## Usage

```bash
node 0-javascript_is_amazing.js
node 1-multi_languages.js
```

## Features

- Basic `console.log` output
- Simple script structure
- Practice with Node.js entry-point files

## Documentation Standard

The scripts should keep short module comments and clear naming for any variables or helpers.

## Contributions & Thanks
